Detailed Description

defines a rail car.

this class inherits the TrainComponent class

The car does not provide power to the train unlike the locomtotive. Cars only cary either commodities or fuel only. if it is fuel tender, it can provide power to the locomotive once the locomotive's stored power is low. In that case, the car losses weight by the fuel consumption.
