Detailed Description

The Simulator class represents a simulation engine for train simulations.

It manages the simulation process, including the trains, time step, output files, network, and progress. The Simulator class handles the execution of the simulation and provides functionality to export trajectory and summary files. It also allows for customization of simulation parameters such as the simulation end time and plot frequency. The Simulator class is derived from QObject and can be used with the Qt framework.
